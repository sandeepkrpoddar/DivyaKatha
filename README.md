## üöÄ Installation

### Prerequisites
- Node.js v18+
- npm v9+ or yarn v1.22+

### Setup Guide
```bash
# Clone repository
git clone https://github.com/[your-username]/divyakatha.git
cd divyakatha

# Install dependencies
npm install --force

# Start development server
npm run dev
```
Access at: `http://localhost:5173`
